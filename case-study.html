How It Was Built

The Agent Pipeline

Stage 1: Evidence Collection
Tavily API searches for official docs. Firecrawl extracts content. Rule-based parser identifies auth method, API type, and self-serve flag. No hallucination, only extracted facts.
Stage 2: Independent Verification
Second pass verifies each claim against evidence URLs. Flags conflicts. Prevents confident fabrication.
Stage 3: Confidence Scoring
Deterministic scoring (0-100): evidence quality + completeness + verification agreement. Not another LLM call.
Stage 4: Pattern Analysis
Cross-app clustering: auth distribution, self-serve vs gated, blockers, ease of integration. Surfaces actionable insights.
Where humans stepped in:
  • Classified 12 ambiguous apps manually (custom auth, unclear docs)
  • Verified sample of 18 findings by reading live docs (accuracy check)
  • Wrote category summaries for each app (one-liner descriptions)
