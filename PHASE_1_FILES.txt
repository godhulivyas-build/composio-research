EXECUTION FLOW

     ├→ loadEnvironment()

     │    │   ├→ validate ResearchFindingSchema
     │    │   └→ collect or skip on error
     │    └→ return array of ResearchFindings
     ├→ ResearchFindingsSchema.parse()
     ├→ writeJsonFile('output/research.json')
     └→ log summary
