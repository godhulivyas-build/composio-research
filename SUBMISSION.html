What Was Built

A 4-stage automated research pipeline that investigates 100 SaaS applications and identifies which can become Composio toolkits:

Stage 1: Research - Tavily API searches for official documentation. Firecrawl extracts content. Parser identifies auth method, API type, self-serve flag, and blockers.

Stage 2: Verification - Independent verification of each claim against evidence. Flags conflicts. Prevents hallucination.

Stage 3: Scoring - Deterministic confidence scoring (0-100). No additional LLM calls. Pure calculation.

Stage 4: Analysis - Cross-app pattern clustering. Auth distribution, access model, blockers, easy wins.
