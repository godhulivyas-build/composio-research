================================================================================
HOW IT WORKS NOW
================================================================================

1. SEARCH STAGE (Tavily)
   ├─ Input: App name (e.g., "Slack")
   ├─ Search: "Slack official documentation API docs authentication"
   ├─ Filter: Keep only official sources
   ├─ Output: Top URLs for official documentation
   └─ Fallback: Hardcoded docs for Slack, Stripe, HubSpot (testing)

2. EXTRACT STAGE (Firecrawl)
   ├─ Input: Documentation URLs
   ├─ Fetch: Each URL's content
   ├─ Clean: Remove nav, ads, noise
   ├─ Format: Convert to clean markdown
   ├─ Truncate: Keep first 8000 chars if needed
   ├─ Output: DocumentationPage[] with clean markdown
   └─ Fallback: Placeholder markdown (testing)

3. PREPARE STAGE
   ├─ Input: DocumentationPage[]
   ├─ Format: Combine all docs into single context
   ├─ Annotate: Label each source URL
   ├─ Output: Documentation string for Claude
   └─ Instruction: "Extract ONLY from above"

4. EXTRACT WITH CLAUDE
   ├─ Input: Documentation string + app name
   ├─ System Prompt: "You are evidence-extraction specialist"
   ├─ User Prompt: Shows documentation + extraction task
   ├─ Constraint: "Use ONLY the documentation provided"
   ├─ Output: JSON with extracted fields
   └─ Validation: Zod schema ensures structure

5. VALIDATE & WRITE
   ├─ Input: Claude JSON response
   ├─ Parse: JSON parse with error handling
   ├─ Validate: Zod ResearchFindingSchema
   ├─ Enrich: Add timestamp
   ├─ Output: output/research.json
   └─ Status: Log summary (total, successful, failed)

================================================================================
KEY BEHAVIORS
================================================================================

When Official Docs NOT Found:
  → integrationBlocker: "Official documentation not found"
  → All fields: null
  → Still creates ResearchFinding (not error)

When Extraction Fails:
  → Skip that URL, try others
  → Partial results collected
  → Pipeline continues to next app

When Claude Doesn't Know:
  → Returns null for that field
  → Explicitly marked as "Unknown"
  → Not marked as found elsewhere

When Everything Succeeds:
  → ResearchFinding with all discovered fields
  → evidenceUrls populated with source docs
  → Ready for Phase 2 Verification
