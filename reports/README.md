# Reports

Generated HTML dashboards and reports.

## Files

- `index.html` - Main interactive dashboard with charts and searchable table
- `methodology.html` - Documentation of research methodology
- `README.md` - Markdown report for documentation

## Features

Dashboard includes:

- Summary metrics (total apps, average confidence, pass rates)
- Interactive charts (authentication methods, API types, category distribution)
- Searchable and sortable table of applications
- Confidence score visualization
- Integration readiness analysis
- Common blockers summary

## Generation

Reports are generated by running:

```bash
npm run generate-report
```

This processes `output/final_findings.json` and `output/analysis.json` into the HTML dashboard.
